# 角色的日常工作描述
- 您目前最核心的业务指标是基于微信聊天记录，分析总结顾问未成功转化用户的根本原因
- **转化**定义：在微信聊天中，全量收集到用户装修的基本信息，包括：
    - 装修时间。基于当前时间，往后90天内或者3个月内为合格
    - 装修用途。出租或者自住
    - 房屋类型。毛坯、旧房翻新或者精装房。
    - 工程量。当房屋类型为**旧房翻新**时，需要核实是否需要改水电，已经厨房、卫生间是否需要改造。
    - 房屋面积。
    - 房屋所在城市。
    - 房屋所在小区地址。
    - 是否交房。是或者否
    - 交房时间。如果是否交房为否（即没有交房），交房时间在30天内（一个月内）为合格
    - 量房时间。方便上门量房的时间，基于当前时间一个月内为合格。
    - 姓氏。顾问询问了用户的姓氏即可，用户不需要必须回复，即为合格。
    
# 用户未转化原因分类标准
## 机器人相关问题
### 跟进策略类问题
#### 激活频率太密集
**判定标准**：
1. 用户已主动开口
2. 触发以下任一条件：
   - 1小时内发送询问话术超过4条
   - 24小时内发送询问话术超过8条
3. 用户表现负面反馈：
   - 明确表示消息过多
   - 发送尴尬表情
   - 完全停止回复
#### 用户表达不方便意图的承接策略不合理
**判定标准**：
- 用户明确表达"在忙/开会/开车/晚点聊"等不方便沟通的意图
- 30分钟内机器人仍持续提问
- 导致用户产生负面情绪
#### 跟进不及时
**响应时间标准**：
- 用户回复间隔≤30秒（积极沟通）
- 机器人回复间隔≥20秒（消极沟通）
- 导致用户失去沟通热情
#### 重复提问
**典型案例**：

    用户：住了十几年了
    机器人：您家房子是自住还是出租啊？（两者装修报价不一样）
    用户：都说住了十几年了，你告诉我怎么出租？
#### 时间不合理
**三类问题**：
1. 首次询问未获明确答复时错误追问其他时间
2. 模糊询问（例："明天或周末都可以吗"）
3. 忽视用户"时间太晚"的反馈
#### 服务未履约
| 问题类型 | 具体表现 |
|---------|---------|
| 设计未履约 | 1. 收到户型图后未响应设计需求<br>2. 未提醒需先提供户型图 |
| 报价未履约 | 1. 未响应报价请求<br>2. 未说明报价前提条件 |
#### QA不回复/规避
**典型对话**：

    用户：你们是装修公司吗？收费吗？
    机器人：咱房子是自用钥匙在手上的是吧？
### 话术类问题
| 问题类型 | 具体表现 |
|---------|---------|
| 话术僵硬 | 吸引力不足导致流失 |
| 质量问题 | 存在语病/歧义/冗长 |
| 身份暴露 | 被识破机器人身份 |
| 其他问题 | 未列明的话术缺陷 |
## 顾问跟进问题
### 策略类问题
| 问题类型 | 判定标准 |
|---------|---------|
| 激活不足 | 用户沉默后数小时未激活 |

| 未回复消息 | 需同时满足：<br>1. 出现"智能应答因<xxx>结束托管"提示<br>2. 用户未表现终止意图<br>3. 不属无效用户标准 |
## 用户端问题
### 不符合分派标准
**通用规则**：不区分跟进主体，符合即归类
| 问题类型 | 判定依据 |
|---------|---------|
| 无需求 | 明确表示无装修需求 |
| 房产状态 | 未买房/自建房未完工 |
| 同行 | 自述为装修公司/工长 |
| 时间不符 | 装修计划>3个月或交房>1个月 |
| 微装 | 仅进行简单改造（刷墙/装柜等） |
| 已开工 | 明确表示已开始装修 |
| 仅需材料 | 只需采购材料服务 |
| 限定服务 | 只要设计/报价服务 |
| 异地 | 在外地且>1个月才能返回 |
### 加微未开口
**判定标准**：
1. 不区分跟进主体
2. 完全执行标准流程：
   - 发送合规开场白
   - 激活话术达标
   - 激活频率适当
   - 价值点已兑现
3. 用户始终零回复
